# Trench-Face-Thai-patch
จาก{ผู้แปล}
-สวัสดีเพื่อนพี่ๆน้องๆที่ได้เห็นข้อความนี้นะครับผมเป็นนักศึกษาหัดแปลเกมเพื่อพัฒนาฝีมือและสกิวของตัวเองเพราะฉะนั้นแล้วตัวภาษาไทยอาจจะมีผิดๆถูกๆบ้าง
ต้องขออภัยด้วยครับผลงานชิ้นนี้ทำด้วยใจครับคุณจะไม่เห็นเครดิตของตัวผมในชิ้นงานเลยผมแค่ทำเพราะรู้สึกว่าเกมนี้ผมเล่นและชอบจึงอยากทำออกมาให้เต็มที่ที่สุดเท่าที่ผมในตอนนี้จะทำได้
ตัวไฟล์ที่ให้เป็นPatchนะครับคุณสามารถซื้อเกมจากต้นทางอย่าง'ตัวเกมหลัก'เพื่อนำไฟล์ของผมลงไปวางทับเพื่อทำให้เกมเป็นภาษาไทยที่สุดครับ

ตัวเกมหลัก(สนับสนุนผู้พัฒนา)

3.เข้าเล่นเกมได้ตามปกติ
